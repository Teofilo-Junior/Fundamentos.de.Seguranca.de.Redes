📘 # MINIGUIA DE ESTUDO — SEGURANÇA DE REDES
📌 1. Contexto e Objetivo da criação do MINIGUIA
Este projeto tem como objetivo estudar os fundamentos de Segurança de Redes utilizando o NotebookLM como ferramenta de apoio.
Mais do que apenas compreender o conteúdo, o foco foi desenvolver um processo de aprendizado baseado em:
- uso estratégico de Inteligência Artificial
- engenharia de prompts
- análise crítica das respostas geradas
- construção de um material de revisão reutilizável

🎯 ## 2. Objetivos de estudo
- Compreender os fundamentos da segurança de redes
- Identificar principais ameaças e vulnerabilidades
- Entender mecanismos de proteção (firewall, criptografia, etc.)
- Aplicar conceitos em contextos práticos
- Desenvolver pensamento crítico no uso de IA

📊 3. Resumos estruturados
🔹 O que é Segurança de Redes
Conjunto de práticas e tecnologias voltadas à proteção de redes contra acessos não autorizados, ataques e falhas.

🔹 Princípios básicos
- Confidencialidade
- Integridade
- Disponibilidade

🔹 Principais ameaças
- Malware
- Phishing
- DDoS
- Sniffing

🔹 Mecanismos de proteção
- Firewalls
- Criptografia
- Autenticação
- Monitoramento

📚 4. Glossário
- Firewall → sistema que controla o tráfego da rede
- Malware → software malicioso
- DDoS → ataque de sobrecarga de servidores
- Criptografia → proteção de dados por codificação
- Phishing → tentativa de roubo de dados por engano

🧠 🔧 6. Prompts reutilizáveis (baseados no curso Técnicas de Engenharia de Prompt - DIO)

🔹 1. Prompt completo (estrutura ideal)
👉 Baseado em: componentes de um prompt

-- Use quando quiser respostas boas desde a primeira tentativa. --

"Explique o conceito de [ex: tema de segurança de redes] de forma clara para iniciantes.
Contexto: estou estudando segurança de redes em nível introdutório.
Inclua:
- definição simples
- exemplo prático
- possíveis riscos ou aplicações
Formato: responda em tópicos organizados."

✔️ Esse é um “prompt padrão”.

🔹 2. Prompt com foco em clareza (Instruções claras)
👉 Baseado em: instruções claras

"Explique o que é [ex: firewall] de forma simples, como se eu não tivesse conhecimento técnico.
Use linguagem fácil e exemplos do dia a dia.
Evite termos complexos sem explicação."

✔️ Ótimo para começar um assunto.

🔹 3. Prompt com divisão de tarefa
👉 Baseado em: dividir tarefa

"Explique [ex: segurança de redes] dividindo em etapas:
1. O que é
2. Para que serve
3. Como funciona na prática
4. Exemplos reais"

✔️ Gera respostas organizadas e completas

🔹 4. Prompt com estrutura de saída
👉 Baseado em: formato de saída

"Explique [ex:ataques de rede.]
Formato da resposta:
- Definição
- Tipos principais
- Exemplo prático
- Como se proteger"

✔️ Ideal para miniguia/conhecimentos gerais sobre um tema específico

🔹 5. Prompt com pensamento guiado (cadeia de raciocínio)
👉 Baseado em: cadeia de pensamento

Explique [ex: como um ataque DDoS acontece passo a passo, desde o início até o impacto final na rede.]

✔️ Excelente para mostrar entendimento profundo

🔹 6. Prompt com comparação
👉 Baseado em: análise crítica

"Compare [ex: firewall e antivírus:]
- principais diferenças
- quando usar cada um
- limitações de cada abordagem"

✔️ Mostra pensamento crítico (muito valorizado)

🔹 7. Prompt com restrições (guardrails)
👉 Baseado em: restrições e limitações

Explique [ex: segurança de redes em até 150 palavras.]
- Foque apenas nos conceitos mais importantes e evite detalhes avançados.

✔️ Bom para resumo e revisão

🔹 8. Prompt com exemplo aplicado
👉 Baseado em: conteúdo de suporte

"Explique como [ex: proteger uma rede Wi-Fi doméstica.]
Inclua:
- práticas recomendadas
- erros comuns
- exemplo real de configuração segura"

✔️ Conecta teoria com prática

🔹 9. Prompt para aprofundamento técnico
👉 Baseado em: variação de nível

"Explique [ex: criptografia em redes] de forma mais técnica.
Inclua:
- como funciona
- onde é aplicada
- limitações"

✔️ Mostra evolução no uso da IA

🔹 10. Prompt para análise crítica (diferencial do seu trabalho)
👉 Baseado em: pensamento crítico

"Quais são as limitações das técnicas de [ex: segurança de redes?]
- Em quais situações elas podem falhar?"
